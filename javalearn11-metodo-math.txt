Método Math
------------
    A classe Java Math possui muitos métodos que permitem realizar tarefas matemáticas em números.

        Math.max(x,y)
        --------------
            O método Math.max(x, y) pode ser usado para encontrar o maior valor de x e y:

                Math.max(5, 10);

        Math.min(x,y)
        --------------
            O método Math.min(x, y) pode ser usado para encontrar o menor valor de x e y:

                Math.min(5, 10);

        Math.sqrt(x)
        -------------
            O método Math.sqrt (x) retorna a raiz quadrada de x:

                Math.sqrt(64);

        Math.abs(x)
        ------------
            O método Math.abs (x) retorna o valor absoluto (positivo) de x:

                Math.abs(-4.7);

        Números aleatórios
        -------------------
            Math.random() retorna um número aleatório entre 0,0 (inclusivo) e 1,0 (exclusivo):

                Math.random();

        Para obter mais controle sobre o número aleatório, por exemplo, você deseja apenas um número aleatório entre 0 e 100, você pode usar a seguinte fórmula:

                int randomNum = (int)(Math.random() * 101);  // 0 to 100
